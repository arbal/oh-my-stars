mystarspilot
=============

Search Github Repositories starred by me through the command line.


```
$ brew install kyoto-cabinet
$ pip install mystartspilot
$ mystars --help
$ mystars --update
$ mystars angular upload
$ mystars --language python
$ mystars awesome python
```
